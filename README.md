# Metro Ticket Booking System

A ServiceNow-based Metro Ticket Booking application that allows passengers and students to book metro tickets, apply for student passes, make payments, generate QR tickets, and raise complaints through a self-service portal.

## Features

### Passenger Features
- Book Metro Tickets
- Fare Calculation
- Online Payment
- QR Ticket Generation
- View My Tickets

### Student Features
- Student Verification
- Apply for Student Pass
- Pass Renewal
- Payment Processing
- View My Passes

### Complaint Management
- Raise Complaints
- Track Complaint Status
- View Complaint History

### Admin Features
- Dashboard Analytics
- Booking Management
- Student Verification
- Revenue Tracking
- Complaint Monitoring

## Technologies Used

- ServiceNow App Engine Studio
- Flow Designer
- Service Portal
- Business Rules
- Script Includes
- Notifications
- Access Controls (ACLs)
